Original URL: [https://codepen.io/matt-cannon/pen/XJKKzPQ](https://codepen.io/matt-cannon/pen/XJKKzPQ).

A neon CRT paddle duel built with Canvas 2D and adaptive AI that ramps difficulty over the course of a match. Gameplay is layered over a stylized vapor backdrop with animated glow fields, RGB fringing, vignette, and scanlines to simulate a phosphor display. Includes responsive mouse, touch, and keyboard controls, performance-capped particle trails and spark bursts, cinematic match-point slow motion, optional Web Audio sound effects, and a Press Start 2P UI with live scoring, final match stats, and replay flow. Works best on a smaller window in full screen mode
